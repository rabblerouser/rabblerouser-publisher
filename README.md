# rabblerouser-stream v0.1.0

Rabble Rouser's simplified events streaming client

## Installation

Using npm:
```shell
$ npm i -g npm
$ npm i --save git+ssh://git@github.com:rabblerouser/rabblerouser-stream.git
```

## Usage
```js
// Load the full build.
const stream = require('rabblerouser-stream');
```
